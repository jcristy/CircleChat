Distribution

Direct other user to [your IP Address]:8082 in a web browser when you are running the client. Their browser will download the CircleChat.jar to their computer.

Connect

  • Click Join
  • Select Method
    • Leech onto - You will not be added into the circle, but incoming and outgoing messages will be forwarded through the user you specify.
    Automatic - You will be added in the circle between the user you specify and his next hop.
    Manual - Your next hop will simply be set. (Not Recommended)
  • Click Connect
    •

    Settings

    Show Time A time stamp will appear next to the message. This time is the time the message was sent.
    Run In Background The program will continue to run and can be accessed from the taskbar. Incoming messages will be counted by a number overlaying the icon.

    Glossary


    LeechA node that is not in the circle, useful for nodes behind firewalls blocking incoming connections
    NodeA user who is part of the ring of message handlers
